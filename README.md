# PeerTutor

Prototipo funcional en React para una plataforma de tutorias peer-to-peer.

## Flujos

- Pantalla inicial con seleccion de rol: estudiante, tutor o administrador.
- Cada rol tiene su propio header, sidebar e inicio.
- Estudiante: busqueda con filtros, perfil de tutor, solicitud, mis tutorias, recursos y mensajes.
- Tutor: solicitudes recibidas, aceptacion/rechazo, horarios, materias y recursos.
- Administrador: resumen, usuarios, perfiles, reportes y normas.

## Tecnologias

- React + Vite.
- Tailwind CSS.
- Componentes locales con estilo shadcn/ui: Button, Card, Badge, Input, Select, Textarea.
- Datos simulados en memoria, sin backend.

## Uso

```bash
npm install
npm run dev
```
